## 0.1.0.beta.6 업데이트 (26.06.09)


### normalize 모드 기본 옵션으로 변경

이제 URLNormalize 가 기본적으로 켜지게 됩니다.

```ts
{
    ...
    URLNormalize : DenchURLNormalizeMode.BOUNDARY,
}
```
따라서 앞으로 `BoundaryNormalize()` 함수를 사용하지 않아도 기본 적용을 할 수 있습니다.


### DenchURLNormalizeMode와 URLNormalize : (normalizeMode : DenchURLNormalizeMode) 추가

```ts

    export enum DenchURLNormalizeMode {
        NONE,
        BOUNDARY,
        HARD
    }

    URLNormalize : (normalizeMode : DenchURLNormalizeMode)=> R

```
이제 dench의 url normalize 설정을 URLNoramzlize에서 enum 값의 설정을 통해서도 설정이 가능해집니다.

기본값은 BOUNDARY 지만, NONE으로 맞추면 정규화를 무시할 수 있습니다.
